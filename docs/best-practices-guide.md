# Best Practices Guide

The HPE Best Practices Guide (BPG) outlines recommended configurations.

__Highlights__

- Enable multipathing for iSCSI and FC.
- Use thin provisioning by default.
- Separate CPGs for different workloads.
- Apply QoS for performance-sensitive applications.
- Use Peer Persistence for high availability.

📄 Full document: [HPE BPG PDF](https://www.hpe.com/psnow/doc/4AA6-5209ENW)
