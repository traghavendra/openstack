# Supported Operations

- Volume create, delete, attach, detach
- Snapshot create, delete, revert
- Create volume from snapshot or image
- Clone volume
- Extend volume
- Manage / unmanage volumes & snapshots
- Thin provisioning
- Deduplication and compression
- QoS and extra-specs
- Consistency groups and group snapshots
- Replication (sync & async)
- Peer Persistence (transparent failover)

**Not supported**
- Multi-attach in clustered mode
